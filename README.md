# Personal Finance Tracker 💰

A beautiful web-based finance tracker to manage your income and expenses.

## Live Demo
> Open `index.html` in any browser — no installation needed!

## Features
- ✅ Add income & expenses
- ✅ 6 categories (Food, Transport, Shopping, Health, Education, Other)
- ✅ Doughnut chart — spending by category
- ✅ Monthly filter — view any month separately
- ✅ Data saved in browser (localStorage)
- ✅ Delete individual transactions
- ✅ Dark theme UI

## How to Use
1. Download or clone this repo
2. Open `index.html` in browser
3. Add your transactions
4. View your spending chart & monthly summary

## Built With
- HTML5
- CSS3
- JavaScript (Vanilla)
- Chart.js
